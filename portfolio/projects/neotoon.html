NeoToon

Une application web dynamique (non publié)

Projet réalisé pour l'UE (unité d'enseignement) "Gestion de projet".
Pour le projet de cette UE nous devions réaliser toute l'arborescence d'une gestion de projet en groupe comme si nous étions dans le monde professionnel. Nous étions par groupe de 3 à 4 étudiants et nous devions désigner un chef de projet. On a également réaliser un cahier des charges, pour attribuer à chaque membre du groupe une ou plusieurs tâches à réaliser pour mener à bien l'idée que nous avions.

On a donc ensemble décidé, en se basant sur un site qui existait déjà de réaliser un site de webtoon. Pour le nom NeoToon il nous est venu l'idée de Neo pour nouveau et Toon car il présenterait des webtoon.
Le site à entièrement été réalisé à l'aide du framework Symfony et nous utilisions git pour la répartitions des tâches. Le site n'est pas à regarder pour son visuel mais plutot pour sa complexité.

Il permettait donc à l'utilisateur de pouvoir se connecter pour regarder, commenter et noter les webtoon. Il y avait également une partie administrateur pour pouvoir gérer les webtoon et les utilisateurs. Nous avons également réalisé toute une partie pour les webtoonistes pour qu'ils puissent poster leurs webtoon. Et enfin, il y avait un forum pour que les utilisateurs puissent discuter entre eux ainsi qu'une partie commentaire sous chaque webtoon.
