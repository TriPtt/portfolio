La Favola

Un site de E-commerce (non publié)

Un projet réalisé dans le cadre de l'université
Ce projet est visuelement le même que le site internet que j'ai publié sur internet, cependant il à été réalisé en liant le framwork Symfony qui a permis de réaliser une API pour la partie back-end et VueJs pour la partie front-end. Il permet à l'utilisateur de se connecter, de s'inscrire, de voir les produits, et de les ajouter dans un panier.
Il y a également une partie administrateur pour pouvoir gérer les produits, les utilisateurs et les commandes.

Le site ayant été effectué dans le cadre de l'université, il n'est pas publié sur internet.
Mais il pourrait être un prototype pour un site de e-commerce, et pourquoi pas remplacer le site actuelle de la pizzeria.
